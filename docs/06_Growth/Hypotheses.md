# Growth Hypotheses

> Структура: **гипотеза → как проверим → метрика → статус**.
> Статусы: `idea` | `running` | `validated` | `rejected`.

## H1 · Pack Call как виральный driver
**Если** в push-приглашении ответить на Pack Call показать сам вопрос («Куда сходить на ДР в Тбилиси?»), **то** CTR на установку будет в 2× выше шаринга стандартного поста.
- Метрика: install rate из shared-link Pack Call vs обычный пост
- Когорта: первые 200 шеров каждого типа
- Статус: `idea`

## H2 · TrustCoins welcome bonus
**Если** новому юзеру при регистрации по приглашению давать 50 TrustCoins, **то** D7 retention будет +30% относительно органики.
- Метрика: D7 retention для invite-cohort vs organic
- Статус: `idea`

## H3 · Public TrustScore = социальное давление
**Если** показывать TrustScore публично на профиле, **то** частота постов вырастет (геймификация).
- Риск: накрутки и обиды у тех чей score низкий.
- Метрика: posts per user per week, до/после.
- Статус: `idea` — после стабилизации формулы (см. [[../04_Features/TrustScore]])

## H4 · «Топ pack недели» — Push
**Если** раз в неделю слать push «топ-3 рекомендации твоей стаи за неделю», **то** WAU вырастет на 15%.
- Метрика: open rate push, return DAU.
- Статус: `idea`

## H5 · Локальный лендинг города
**Если** под каждый город делать отдельный landing с топ-постами/коллекциями, **то** organic SEO даст приток.
- Метрика: organic installs из этих лендингов.
- Статус: `idea`

## Бэклог идей
- Реферальная программа с разделением TrustCoins (юзер + приглашённый получают)
- «Найди свою стаю» — match по интересам и геолокации
- Экспорт коллекции в Instagram Stories (готовый шаблон)
