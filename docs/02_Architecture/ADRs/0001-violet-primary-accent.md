# ADR-0001: Violet (#7C6FE0) как primary accent вместо Lime

## Контекст
Изначально `AppLime` был основным акцентом по системе Artisan Pastel. На практике Lime:
- терялся на тёплом off-white фоне (`#F8F7F4`),
- слабо контрастировал с Teal в градиенте (получался «кислотный» эффект),
- плохо смотрелся с Gold (`AppGold`) — двух «жёлтых» акцентов рядом.

Нужно было выбрать акцент, который:
1. Контрастирует с фоном.
2. Образует читаемый градиент с `AppTeal`.
3. Не конфликтует с золотым цветом монетизации.

## Решение
`AppViolet = #7C6FE0` — primary accent.
Градиент кнопок и активных элементов: `AppViolet → AppTeal` (`#7C6FE0 → #3BD4C0`).
`AppLime` оставлен как **alias на AppViolet** для обратной совместимости в коде, чтобы не переписывать сразу всё.

## Альтернативы
1. **Coral / тёплый красный** — отвергли: слишком «эмоциональный», давил на нейтральный pastel-стиль.
2. **Deep teal** — отвергли: слился бы с `AppTeal`, исчез бы градиент.
3. **Оставить Lime + поднять контраст фона** — отвергли: разрушило бы тёплый off-white characte.

## Последствия
**+ Плюсы:**
- Читаемый контраст на фоне.
- Гармоничный градиент Violet→Teal — выглядит «премиально».
- Чёткое разделение: Violet = интерактив, Gold = монетизация, Teal = акцент.

**− Минусы:**
- Alias `AppLime = AppViolet` создаёт когнитивный диссонанс при чтении кода. Когда-то надо вычистить.
