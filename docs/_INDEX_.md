# 🧭 System Index: Recommend

## 🎯 Суть (Vision)
Анти-рекламная экосистема локальных рекомендаций. Мы заменяем проплаченные отзывы Google честным социальным графом (Pack) и вознаграждаем доверие через TrustCoins.

## 🛠 Технологический стек
- **IDE:** Cursor (основная), Android Studio.
- **Frontend:** Kotlin, Jetpack Compose.
- **Backend:** Firebase (Firestore, Cloud Functions).
- **AI Tools:** OpenClaw.ai, Claude API.
- **Design:** Artisan Pastel & Modern Monotone.
